<Info>
This feature is particularly useful to edit the order and groups of pages in your navigation.
</Info>

## Sidebar
To edit your sidebar, make changes to the `navigation` array in your `docs.json` file. For example:

```json Navigation
"navigation": [
  {
    "group": "Getting Started",
    "pages": ["quickstart", "development"]
  },
  {
    "group": "Essentials",
    "pages": ["markdown", "code", "images"]
  }
]
```

## Navbar

For navbar in your documentation, you can add navbar items by editing the `navbar` field in your `docs.json`.

```json navbar
"navbar": {
  "links": [
    {
      "label": "Changelog",
      "href": "/changelog"
    }
  ]
}
```
